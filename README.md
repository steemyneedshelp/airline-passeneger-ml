# Airline Passenger Machine Learning Project

This project explores airline passenger data using machine learning.  
The notebook focuses on data exploration, preprocessing, and applying basic models to analyze the dataset.

## Repository Contents

- `airline_passenger_analysis.ipynb`  
  Main Jupyter Notebook containing the analysis and machine learning steps.

- `test.csv`  
  Dataset used in the notebook.

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib (for visualizations)
